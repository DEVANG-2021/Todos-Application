About Todos Application

  • This is Simple Todos Application which is used to perform Basic operation for daily tasks.
  • Features Provided by the application are :
  • User is able to Add task
  • User is able to Delete task
  • User is able to Search task
  • Tasks are stored in localstorage of browser, so User can see all the tasks in future whenever he or she will visiting web application
